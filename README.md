## Getting Started
To get started with this project, clone this repository.
```
git clone 
```

Install the necessary libraries via pip.
```
pip install -r requirements.txt
```

Running `.py` files can be done by using this command:
```
python <filename>.py
```

Opening and running `.ipynb` files can be done by either through Jupyter Lab or Visual Studio Code.
